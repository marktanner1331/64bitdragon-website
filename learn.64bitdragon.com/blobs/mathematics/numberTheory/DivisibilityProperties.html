Factors of 0
Every number is a factor of 0:

LaTeX: 0 \mid n \implies n = 0
Proof
  1. If dk = n then there must always be a k such that dk = 0
  2. We can set k = 0
0 as a factor
If 0 is a factor of n then n = 0:

LaTeX: 0 \mid n \implies n = 0
Proof
  1. If dk = n and d = 0 then 0k = n
  2. n must be 0
