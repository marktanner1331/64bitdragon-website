How it works

This online tool will allow you to escape and unescape text for use in strings in Java and any c-style language including:
  • C
  • C++
  • C#
  • Javascript
  • Perl
  • PHP

It will replace the following characters:
Name Escaped
Backslash \\
Carriage Return \r
Double Quote \"
Form Feed \f
New Line \n
Tab \t
