Word Count

How it works

This tool counts the number of characters and words in the submitted text

When counting the amount of characters, several different types of characters are counted.
These include: letters, numbers, punctuation, spaces and new lines.
New lines will count as two characters, as they are made of a carriage return followed by a line feed.

When counting words, we count the number of spaces, and then add 1.

The word and character counts are automatically calculated every time the input text is changed.

This online tool for counting words and characters is completely client side and runs in the browser.
